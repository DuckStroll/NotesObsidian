# Definition
Un Corps est un [[Anneaux]] Unitaire $(K,+,\cdot)$ tel que $$a\cdot b = b\cdot a$$
Par exemple: $\mathbb{Z}/p\mathbb{Z}$ est un anneau unitaire pour $p$ premier.
